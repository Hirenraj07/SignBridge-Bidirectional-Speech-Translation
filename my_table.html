Test Case ID Test Description Steps to Perform Expected Output
TC-01 Application Launch (Inference Mode) Run `python main.py` when a model file exists. OpenCV window opens, and camera feed is live.
TC-02 Initial State Verification Observe the text overlay at the bottom of the screen upon launch. The prediction area is empty until the frame buffer is filled.
TC-03 First Prediction Trigger Remain in front of the camera for at least 150 frames (5-10 seconds). The prediction text appears at the bottom of the screen.
TC-04 Continuous Translation Perform a series of different signs from the dataset. The prediction text updates to reflect the new signs being performed.
TC-05 Model Accuracy Test (Known Sign) Perform a clear, simple sign that the model was trained on (e.g., "book"). The predicted text correctly identifies the sign.
TC-06 System Behavior with No User Move completely out of the camera's view. The system does not produce new predictions.
TC-07 Graceful Exit Press the 'q' key while the OpenCV video window is active. The application window closes, and the script terminates without errors.
